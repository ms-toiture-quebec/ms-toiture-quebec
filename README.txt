
MS Couverture Québec – Site statique
Date: 2025-11-10

Comment utiliser
1) Téléchargez l'archive ZIP.
2) Décompressez-la sur votre ordinateur.
3) Ouvrez index.html dans votre navigateur.
4) Remplacez les informations suivantes dans les fichiers HTML:
   - Licence RBQ: RBQ 0000-0000-00
   - NEQ: NEQ 0000000000
   - Téléphone: (418) 555-0123
   - Courriel: info@mscouvreur.ca
   - Adresse: 123 rue de la Toiture, Saint-Raymond, QC G3L 0A0
   - Régions desservies: Portneuf, Québec, Trois-Rivières, Mauricie et environs
5) Mettez vos photos et remplacez les textes au besoin.
6) Hébergez les fichiers chez n'importe quel hébergeur statique (Netlify, Vercel, OVH...).

Notes
- Le bouton "Envoyer la demande" est statique (pas de backend). Pour le rendre fonctionnel, utilisez un service de formulaire (ex: Formspree) ou branchez-le à votre site WordPress.
- Tous les textes emploient le vocabulaire québécois: soumission, bardeaux d’asphalte, toits plats (élastomère), déneigement, CNESST, etc.
- Couleurs principales définies dans style.css (variables CSS).
