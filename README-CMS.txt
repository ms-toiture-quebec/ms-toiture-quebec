MS Toiture Québec – Version avec CMS
Date: 2025-11-10

Pour éditer le site dans le navigateur (ajouter photos, changer téléphone) via /admin :
1) Mets ces fichiers dans un dépôt GitHub (ex: ms-toiture-quebec).
2) Sur Netlify, crée un site "Import from Git" avec ce repo (ou connecte ton site existant au repo).
3) Dans Netlify > Settings > Identity : Enable Identity. Clique "Invite users" et invite ton e-mail.
4) Dans Netlify > Identity > Services : Enable Git Gateway.
5) Déploie. Puis va sur https://TON-SITE.netlify.app/admin et connecte-toi.
6) Dans l'admin, ouvre la page à éditer (Accueil/Services...) et téléverse tes photos (stockées dans images/uploads).
7) Clique "Publish" : le site se met à jour tout seul.
